#  Tennis Lessons Marketplace App

A web application that allows users to browse, book, and pay for tennis 
lessons. Built with Python and Streamlit, and integrated with Stripe for 
secure payments.

---

#  Features

* ‍ Browse available tennis coaches
*  Book lessons
*  Secure payments with Stripe
*  Simple and fast UI with Streamlit

---

#️ Tech Stack

* Python
* Streamlit
* Stripe API
* python-dotenv

# Environment Variables

Create a `.env` file in the root directory but ⚠ never commit your 
`.env` file.

---

#️ Run Locally

1. Clone the repo
   git clone https://github.com/richygamo-ml/Tennis-Marketplace.git

2. Go into the folder
   cd Tennis-Marketplace

3. Install dependencies
   pip install -r requirements.txt

4. Run the app
   streamlit run Tennis_Lessons_App.py
